This is a Vite + React + Tailwind project containing Phase 1 mini-games:
- Match-3 (Bejeweled-like)
- Peggle-inspired
- Word Puzzle
- Tetris Lite
- Memory Match

## Run locally
1. Unzip folder
2. `npm install`
3. `npm run dev`

Open http://localhost:5173

This build uses Tailwind CSS (you may run `npx tailwindcss -i ./src/index.css -o ./dist/output.css --watch` if needed). The project is intentionally minimal and the games are simplified calm versions for prototyping.
